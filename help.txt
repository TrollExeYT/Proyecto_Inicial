Comandos basicos para el proyecto

Creación de ambiente virtual:

python -m venv .venv # Crea el entorno virtual WINDOWS
python3 -m venv .venv # MAC, usar python3 para todos los comandos que digan python

.venv\Scripts\activate # Activa el entorno virtual WINDOWS
source .venv\Scripts\activate # MAC

deactivate # Desactiva el entorno virtual

pip freeze > requirements.txt # Para crear los requerimientos

pip install -r requirements.txt # Para instalar los requerimientos


Django:

django-admin startproject myproject # Iniciar proyecto

python manage.py startapp myapp # Crear una app dentro del proyecto

python manage.py runserver # Iniciar servidor del proyecto, con ctrl+c en el cmd se cierra el servidor

python manage.py createsuperuser # Sirve para crear un usuario con permisos de administrador

# Migrar base de datos, se debe hacer en el siguiente orden
1. python manage.py makemigrations

2. python manage.py migrate


Git:

git pull # Revisar actualizaciones

git add . # Añadir todos los cambios

git commit -m "MENSAJE IMPORTANTE" # Registrar con un mensaje

git push # Meter pal repositorio

git checkout * # USAR SOLO EN CASO DE EMERGENCIA, restaura todos los archivos hasta el ultimo commit
